# Managing Your Addresses

This guide will help you manage your shipping addresses on our website.

## Adding a New Address

1. Go to the "Your Account" page.
2. Click on the "Your Addresses" section.
3. Click the "Add New Address" button.
4. Fill out the address form and click "Save Address".

## Editing an Address

1. Go to the "Your Addresses" section.
2. Find the address you want to edit and click the "Edit" button.
3. Make your changes in the form and click "Save Address".

## Deleting an Address

1. Go to the "Your Addresses" section.
2. Find the address you want to delete and click the "Delete" button.
3. Confirm that you want to delete the address.

## Setting a Default Address

When you add or edit an address, you can check the "Set as default address" box. This address will be automatically selected during checkout.
